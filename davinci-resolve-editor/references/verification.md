# Delivery Verification

1. Confirm active project and timeline names match the approved version.
2. Count video/audio/subtitle items per track.
3. Derive timeline origin from real items.
4. Scan all boundaries for gaps and overlaps.
5. Validate source path, source in/out, and media duration.
6. Calculate duration by source group/camera.
7. Flag adjacent reuse of the same source.
8. Confirm authorized LUT/property writes by readback.
9. Confirm media is online.
10. Save the project and produce a report tied to this timeline identity.
11. Export clip ranges, source identity, captions, and narration/dialogue
    placement needed for a script-to-cut missing-coverage audit.
