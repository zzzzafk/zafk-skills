## Playback (All Pages)

| Shortcut | Action |
|----------|--------|
| Space | Play/Stop |
| J | Reverse Play |
| K | Stop |
| L | Forward Play |
| J+K or K+L | Slow shuttle (1/2 speed) |
| ← / → | 1 frame back/forward |
| Shift+← / Shift+→ | 1 second back/forward |
| ↑ / ↓ | Previous/Next edit point |
| Home / End | Timeline start/end |
| I | Mark In |
| O | Mark Out |
| Alt+X | Clear In & Out |
| Shift+I / Shift+O | Jump to In / Out |
| M | Add Marker |
| Ctrl+/ | Loop Playback |
| \ | Play around current position |
